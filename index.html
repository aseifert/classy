Classy Tutorial

Fast Development

Whether for prototyping or production, reduce the time to get things up and running by leaps and bounds. From data to deployed systems in a matter of few commands.

CLI Is All You Need

Classy is built around the command line interface, you can train, present and REST-expose powerful ML models without writing a single line of code just by executing bash commands.

Data Centric

In Classy data are central. Organize your datasets in common formats like TSV and JSONL and Classy will do the rest.
